Messaging
Web App

UI Design | Front-end | Back-end

The aim of this project was to build a web-application with a chat function for a university project. This web-application allows users to add contacts, chat with friends and create group chats. The project was built by myself and three other students.

The Solution

First we created a mockup using Adobe XD to visualize the application. Then, the app was built using the MERN stack, which includes the database MongoDB, Express, the front-end framework React.js and Node.js. Socket.io, a JavaScript library for realtime web applications, was used to enable real time communication between web clients and the server.
